# HDP-Viz
HDP Visualizer for component services

##Requirements
* NodeJS -- https://nodejs.org/
* Grunt
* Bower
* A Working HDP Cluster -- http://www.hortonworks.com


##Build Environment Setup
1. Have NodeJS Installed
2. Install Grunt and Bower
	`npm install -g grunt`
	`npm install -g bower`
